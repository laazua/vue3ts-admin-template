### 

- **创建项目**
```text
1.  npm create vue@latest  (设置项目名: vue3ts-admin-template, 选择： TypeScript)
2.  npm install @vue/tsconfig
3.  npm install -D unplugin-vue-components unplugin-auto-import
4.  npm install vite-plugin-mock --save-dev
5.  npm install axios
6.  npm install element-plus --save
7.  npm install vue-router@4
8.  npm install pinia

9.  npm install terser
10. npm install vite-plugin-javascript-obfuscator -D
```

- **重启Vue**
```text
快捷键: Ctrl + Shift + p
输入指令: restart && 选择 Vue: Restart Vue and TS servers
```
